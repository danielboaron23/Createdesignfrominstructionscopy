# תרגיל 05: פרופיל משתמש

**רמת קושי:** ⭐⭐⭐
**זמן משוער:** 60 דקות
**פוקוס:** Data aggregation, layout, conditional UI

## מטרה

בנה דף פרופיל אישי למשתמש: סטטיסטיקות, היסטוריית משחקים, רצף יומי. הדף יוצג גם למשתמש הנוכחי וגם למשתמשים אחרים (בלי אפשרות לערוך).

## רקע

יש לנו דאטה עשיר על כל משתמש: פרטים בסיסיים (`USERS`), היסטוריית משחקים (`SCORES`), ורצף יומי (`STREAKS`).

## משימות

1. **הוסף `Page`**: `{ kind: "profile"; userId: string }`.
2. **צור `UserProfilePage.tsx`** המציג:
   - **כותרת**: אווטאר (עיגול צבעוני עם emoji), שם תצוגה, @username, עיר, תאריך הצטרפות
   - **תג Premium** אם רלוונטי (`isPremium`)
   - **סטטיסטיקות**: סך משחקים, win rate כולל, ממוצע ניקוד, משחק מועדף
   - **רצף יומי**: 🔥 X ימים (current), best: Y ימים
   - **היסטוריית משחקים**: רשימת 10 המשחקים האחרונים עם תאריך, משחק, ניקוד, win/loss
3. **חישוב נכון** של "המשחק המועדף" — זה שהמשתמש שיחק הכי הרבה פעמים.
4. **טפל ב-empty states**: משתמש בלי שום משחק → "עדיין לא שיחק".

## רמזים

- **Custom hook עוזר:** `useUserProfile(userId)` שטוען user + scores + streak במקביל (Promise.all).
- **Parallel fetches:** הקריאות בלתי-תלויות — `Promise.all([fetchUser, fetchUserScores, fetchUserStreak])`.
- **Sub-agent:** _"מצא בכל הקבצים אילו פיצ'רים יש בכרטיסי משתמש או באווטאר שכבר בקוד"_
- **GameId display:** ה-`GameId` הוא string slug ("five-letters") — בנה מיפוי לשם בעברית: "5 אותיות".
